Utilizando **sin definir** las funciones del ejercicio anterior, te pedimos que crees las siguientes:

* `losTriples`, que toma una lista de números y devuelve una lista con los triples de cada número,
* `lasIniciales`, que toma una lista de nombres (sin apellido) y devuelve una lista con la inicial de cada nombre,
* `mayoresA10`, que toma una lista de números y devuelve una lista que solo incluye a los elementos mayores a 10.
